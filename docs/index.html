Angular Login Component - Module Federation

Bu sayfa GitHub Pages üzerinde host edilen bir Angular Module Federation remote modülüdür.

Remote Entry URL: https://[username].github.io/[repo-name]/remoteEntry.js

Kullanım

Bu modülü React veya başka bir host uygulamadan kullanmak için:

1. Host uygulamanızda Module Federation ayarları:

// webpack.config.js
new ModuleFederationPlugin({
  name: 'host',
  remotes: {
    angularApp: 'angularApp@https://[username].github.io/[repo-name]/remoteEntry.js',
  },
})

2. Component'i import edin:

// React örneği
await import('angularApp/LoginComponent');

3. Custom element olarak kullanın:

<angular-login-form></angular-login-form>
